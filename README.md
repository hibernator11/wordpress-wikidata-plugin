# wordpress-wikidata-plugin

This repository is an examples of Wordpress plugin to retrieve information from Wordpress.

This code has been employed for teaching purposes (Sistemas de gestión de contenidos y usuarios en la Web) at the University of Alicante since 2020. The students have to adapt the code to their content.
